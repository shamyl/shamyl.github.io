LearnOSTEAM is our flagship EdTech platform designed specifically for STEAM education in schools.

## Features

- **Interactive Lessons**: Engaging content covering Science, Technology, Engineering, Arts, and Mathematics
- **Coding Environment**: Built-in code editor with support for block-based and text-based programming
- **Progress Tracking**: Real-time analytics for teachers and parents
- **Curriculum Aligned**: Content mapped to local and international standards

## Tech Stack

- React.js frontend
- Node.js backend
- PostgreSQL database
- AWS infrastructure

## Impact

Currently being used by schools across Pakistan, helping thousands of students learn coding and robotics concepts.
