CodiBot is an affordable robot kit we designed specifically for educational purposes in developing countries.

## The Problem

Traditional educational robots are expensive, making them inaccessible for most schools in Pakistan and similar markets.

## Our Solution

We created a robot that:

- **Uses 3D printing** for affordable, replicable parts
- **Open-source design** allowing local manufacturing
- **Arduino-based** for easy programming
- **Modular construction** for teaching assembly

## Features

- Line following
- Obstacle avoidance
- Wireless control via Bluetooth
- Programmable via block-based and Python interfaces
